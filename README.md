# Process Simulator

Интерактивный симулятор процесса проработки задач. Визуализирует движение задач по доскам Refinement и Work через 37 кейсов, сгруппированных в 4 волны.

## Волны кейсов

| Волна | Кейсы | Фокус |
|-------|-------|-------|
| 1 | #1–#12 | Базовые сценарии: fast track, investigation, refinement, техдолг |
| 2 | #13–#22 | Edge cases: возвраты, блокировки, эскалации, scope change |
| 3 | #23–#31 | Деградация и масштабирование: SPOF, параллельные консилиумы, инциденты |
| 4 | #32–#37 | Fallback, замещение ролей, ретроспектива, эволюция процесса |

## Использование

Открыть [GitHub Pages](https://kshvakov.github.io/process-simulator/) или локально:

```
open index.html
```

Выбрать кейс → читать описание → нажимать «Далее» (или стрелку →) и наблюдать как карточка двигается по доскам.

## Управление

- **Далее / ←** — следующий шаг
- **Назад / →** — предыдущий шаг
- **Автоплей** — автоматическое воспроизведение
- **Сброс** — вернуть доски в исходное состояние
